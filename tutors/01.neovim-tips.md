# NeoVim tips


## Basic usage

1. Launch nvim with different configs quickly by setting `NVIM_APPNAME` environment varible:

  ```sh
  NVIM_APPNAME="nv-new" nvim init.lua
  ```
  
  as long as you create the `nv-new` folder in `~/.config` (Linux) or `$env:LOCALAPPDATA` (Windows)


2.  Check vim library table:

  ```lua
  :lua =vim
  ```
  
  Check certain fields:

  ```lua
  :lua =vim.api
  :lua =vim.api.nvim_create_autocmd
  ```

3. Check help/manual:

  ```lua
  :help
  -- or simply
  :h
  ```
  
  Check certain topic: 
  ```lua
  :help nvim_create_autocmd
  ```

4. Check important paths:

  ```lua
  -- check current runtime paths
  :echo nvim_list_runtime_paths()
  ```
  
  Important paths for setting configs:
  ```lua
  -- Get current config directory (where init.lua is)
  local configdir = vim.fn.stdpath("config")

  -- Get current data directory (where plugins are installed)
  local datadir = vim.fn.stdpath("data")
  ```

5. Open native file browser:

  ```lua
  :Explore
  ```


## Lua related

1. You could execute current lua file by:

  ```vim
  :source %
  ```

  or select lines in visual mode and execute:

  ```vim
  :'<'>lua
  ```
  
  if you want to execute something:
  ```vim
  :lua ...
  ```

  For example:
  ```lua
  -- try to execute:
  print("hello neovim")
  print("hello neovim")

  -- try to execute the following function by `:lua MyFunc()`
  MyFunc = function()
    print("hello")
  end
  ```

## Options

1. Check option value:

  ```lua
  :set someopt?
  ```

  Check option manual:

  ```lua
  :help 'someopt'
  ```
  
  False boolean option return with `no` prefix:
  ```lua
  :set relativenumber?
  -- If true, return relativenumber
  -- If false, return norelativenumber
  ```

2. Set option value:

  ```lua
  :set {opt}={value}
  :set boolopt  -- set boolean option to true
  :set boolopt! -- set boolean option to false
  ```


## Keymaps

1. You could set keymaps via:

  ```lua
  vim.keymaps.set(...)
  ```

  Put your keymap configs in your config file (e.g. `~/.config/nvim/init.lua`) to enable your keymaps permanantly.

  For example:
  ```lua
  vim.keymap.set("n", "<space>X", "<cmd>source %<cr>")
  vim.keymap.set("n", "<space>x", ":.lua<cr>")
  vim.keymap.set("v", "<space>x", ":lua<cr>")
  ```

2. Check all current keymaps by:
  
  ```lua
  :nnoremap
  ```

  Search certain key:
  ```lua
  :nnoremap <space>
  ```
  
  after Neovim 0.12, also:
  ```lua
  :map  -- all keys
  :map <space>  -- certain key
  ```


## Auto Commands

1. You can change editor behavior by creating autocmd.

  For example:

  ```lua
  -- Highlight when yanking (copying) text
  --   Try it with `yap` in normal mode
  --   See `:help vim.highlight.on_yank()`
  vim.api.nvim_create_autocmd("TextYankPost", {
    desc = "Highlight when yanking (copying) text",
    group = vim.api.nvim_create_augroup("kickstart-highlight-yank", { clear = true }),
    callback = function()
      vim.highlight.on_yank()
    end,
  })
  ```
